WORK 03

Webサイト制作

綺麗肌図鑑

溢れる美容情報の中で「何が正しいのかわからない」という方のための、 皮膚科学に基づいた正しいスキンケア知識を美しく、 分かりやすくまとめたデジタル図鑑です。

学べること

自分の本当の肌質を導き出す「肌質診断」、化粧品裏面の成分表が読めるようになる「成分図鑑」、 毎日のスキンケアの基本などを網羅して体系的に学ぶことができます。

Tools

HTML5 / CSS3 / JavaScript

KOREA EASY TRIP

「初めての韓国旅行、これ見ればOK。」をコンセプトに、 アプリ、エリア、注意点など、旅行前に本当に知りたい情報をギュッとまとめた旅行ガイドサイトです。

主なコンテンツ

現地で役立つ必須アプリの紹介や、注目の最新エリア情報、 トラブルを防ぐための事的事前の注意点など、実用的なノウハウを詰め込みました。

Tools

HTML5 / CSS3 / Adobe Photoshop
